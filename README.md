#  Quiz app

A basic Quiz web app made with Vanilla js

## Features

- User Signup, Login and User Authentication
- Timer and indicator for remaining time
- Random questions
- Mark question


## Tools



- Vanilla js
- JSON
- Fetch API
- npm


## Installation

First install dependencies:

```bash
npm install
```
Then start with live server if installed
